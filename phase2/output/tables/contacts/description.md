# Contacts

## Summary
This table contains information about contacts associated with various accounts, including personal details and contact information.

## Row meaning
Each row represents a unique contact associated with an account.

## Relationships
This table can be linked to the accounts table using the `account_external_id` field.

## Datasource
Filename: uploads/acme-co_contacts_initial_20260418.csv

## Retrieval process
This dataset can be retrieved from a CSV file named `acme-co_contacts_initial_20260418.csv`. There are no pagination or filtering options as it is a static file.
