# Contacts

## Summary
The Contacts table contains information about individual contacts associated with each customer. It includes details such as contact ID, name, title, email, phone, and department.

## Row meaning
Each row represents a unique contact associated with a customer.

## Relationships
This table is related to the Customers table through the customer ID.

## Datasource
Extracted from the `contacts` sheet of the uploaded Excel file `Div_s_Furniture_Manufacturing_Co.xlsx`.

## Retrieval process
This data is part of the Excel file and can be accessed by reading the `contacts` sheet.
