# Customers

## Summary
The Customers table contains information about the clients of Div's Furniture Manufacturing Co, including their contact details, payment preferences, and account status.

## Row meaning
Each row represents a unique customer.

## Relationships
This table is related to the Contacts table through the customer ID, and to the Invoices table through the customer ID.

## Datasource
Extracted from the 'customers' sheet in the Excel file 'Div_s_Furniture_Manufacturing_Co.xlsx'.

## Retrieval process
The data can be retrieved by accessing the 'customers' sheet in the provided Excel file.
